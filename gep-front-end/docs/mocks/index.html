Order Oasis — Dark-mode Mocks

Complete set of dark-mode UI mocks for the 26 screens of the application. All share one visual language: warm-dark canvas + amber accent, Roboto typography, 4 px spacing rhythm — per the reference screenshot and order_oasis/DESIGN.md.
How to read these mocks
No JavaScript. State variants (loading, empty, error, success, role-gated) appear as HTML comments inline. Numbers, IDs and names are realistic-but-mock — they are not wired to any backend.
